# LangGraph Agents Lab

Exploring Agentic AI architectures using LangGraph and Python.

## Implementations
1. **Basic ReAct Agent:** A cyclic graph that reasons and uses tools.
2. **Robust Error Handling:** Custom tool logic that catches `ZeroDivisionError` and self-corrects.
3. **Memory & Persistence:** Using `MemorySaver` to maintain conversation context across threads.

## Tech Stack
* **Orchestration:** LangGraph
* **LLMs:** Google Gemini 1.5 Flash, OpenAI GPT-4o
* **Tooling:** LangChain Core
